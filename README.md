# PureDNS Filters

### Family Filter

This filter blocks adware + malware + fakenews + gambling + porn.

DoH URL: `https://family.puredns.org/dns-query`

### Security & Adblock Filter

This filter blocks adware + malware.

DoH URL: `https://security.puredns.org/dns-query`
